Wat is Pure?

Wat is het?

Pure is een set van kleine, responsieve CSS modules dat je kan gebruiken in elk web project.

CSS met een minimalistisch footprint

Pure is ontzettend klein. De hele set van modules bevat 3.8KB. Het framework is gemaakt met mobiele devices in gedachten. Het was belangrijk voor de makers van Pure om hun files zo klein mogelijk te houden. Daarnaast is elke CSS lijn zorgvuldig bedacht.

Staat je niet in de weg

Pure heeft minimalistische styles en moedigt je aan om jouw eigen style er boven op te schrijvenn. Het is gemaakt om uit je weg te gaan en maakt het daardoor gemakkelijk om er met je eigen stijl er overheen te gaan.
